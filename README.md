1) Название:«CookBook: Коллекция ваших любимых рецептов»
2) Разработчики: «Ковалева Алиса Александровна", "Курилов Александр Сергеевич»
3) Жанр: «Веб‑приложение, Кулинарный сервис»
4) Год выпуска: «2026»
5) Цель: создать удобный и функциональный веб‑сайт для поиска, 
сохранения и систематизации кулинарных рецептов с возможностью персонализации и взаимодействия пользователей. 
Проект должен обеспечить интуитивно понятный интерфейс для пользователей любого уровня компьютерной грамотности и стать надёжным помощником на кухне.
6) Описание:
CookBook — это современный онлайн‑сборник рецептов, который выходит за рамки простой базы данных. Это ваш персональный кулинарный ассистент.
Пользователь, заходя на сайт, сразу попадает на главную страницу с подборкой популярных и новых рецептов.
Он может зарегистрировать личный профиль, чтобы сохранять понравившиеся рецепты в «Избранное», создавать собственные коллекции 
(например, «Праздничные блюда» или «Быстрые завтраки»), так же пользователь может смотреть ближайшие рестораны в которых готовят данную еду по яндекс карте. 
Сайт призван упростить процесс приготовления еды, помогая пользователям находить вдохновение, новые места,
планировать меню и систематизировать свои кулинарные находки в одном месте.
7) Основные функции:
А. Для неавторизованного пользователя:
Просмотр главной страницы с карточками рецептов (название, фото, краткое описание).
Поиск рецептов по названию, ингредиентам или тегам (например, «вегетарианское», «десерт», «из курицы»).
Фильтрация рецептов по категориям (кухня мира, тип блюда, сложность, время приготовления).
Просмотр детальной страницы рецепта (полный список ингредиентов с количеством, пошаговая инструкция, фото/видео этапов приготовления, советы от автора).

Б. Для авторизованного пользователя (после регистрации и входа в систему):
Личный профиль: страница с информацией о пользователе (имя, аватар) и настройками аккаунта (смена пароля, email).
Избранное: возможность сохранять и удалять рецепты из списка «Избранное».
Пользовательские коллекции: создание, редактирование и удаление собственных коллекций рецептов (папок). Возможность добавлять рецепты в несколько коллекций.
История просмотров: автоматический журнал последних просмотренных рецептов.
Сортировка и фильтрация в профиле: возможность сортировать сохранённые рецепты по дате добавления, алфавиту, популярности; фильтровать по коллекциям.
Добавление своих рецептов:** форма для публикации собственных рецептов с загрузкой фото, указанием ингредиентов, шагов приготовления, времени готовки, сложности и тегов.
Оценка и отзывы:** возможность ставить оценки (например, от 1 до 5 звёзд) и оставлять комментарии к любым рецептам.
Список покупок:** функция, позволяющая автоматически формировать список необходимых ингредиентов из выбранного рецепта или нескольких рецептов. Возможность редактировать этот список (добавлять/удалять пункты).
Печать рецепта:** удобная версия рецепта для печати без лишних элементов интерфейса.

В. Административные функции (для модератора/администратора):**
Модерация добавленных пользователями рецептов перед публикацией (проверка на соответствие правилам, отсутствие спама).
Управление учётными записями пользователей (блокировка при нарушении правил).
Добавление и редактирование основных категорий и тегов.


Технологии: бэкенд — Python (фреймворк Flask), фронтенд — HTML. База данных — SQLite.
Безопасность:** надёжная аутентификация и авторизация.
